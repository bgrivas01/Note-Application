step to use this web app:
0. BackEnd run Requirement: Flask-cors <br />
   to install -> python -> pip install flask-cors <br />
   if there is problem when install on windows, install python first, then "python -m pip install flask-cors" in cmd <br />


1. run App.py to get the backend running <br />
2. run index.html file directly or use live server to run the web app. <br />


This project can also run from vs code by:
1. select and open this folder in vs code. <br />
2. use the terminal in vs code and use "python App.py" <br />
3. use live server or open index.html directly to run this web app. <br />
